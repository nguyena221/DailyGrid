# Daily Grid

Daily Grid is a visual weekly planner that helps users organize tasks into balanced daily schedules with optional AI-assisted planning.

## Features
- Weekly & daily planning views
- Task and event management
- Completion tracking
- AI-generated weekly plans (v1)

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- date-fns

## Status
MVP in progress.
